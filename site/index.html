Features

Document Viewer

Browse stacks and pages with thumbnail previews. Supports PDF, JPEG, TIFF and PaperPort .max files.

Scanning

Scan directly from SANE-compatible scanners with presets for resolution, colour mode, duplex and destination folder.

OCR and Search

Built-in OCR engine with full-text search across your entire document collection.

Search Server

REST API server for querying paper repositories remotely. Converts documents to PDF on the fly.

Mobile App

Flutter app for Android and iOS. Browse directories, search documents and view files from your phone.

Print and Export

Print stacks and pages with annotations. Convert between PDF, JPEG and TIFF formats.
