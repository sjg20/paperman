Installation
============

Ubuntu PPA
----------

On Ubuntu, the easiest way to install Paperman is from the PPA:

.. code:: bash

   sudo add-apt-repository ppa:sjg1/ppa
   sudo apt update
   sudo apt install paperman

This provides pre-built packages for the following Ubuntu releases:

- Bionic (18.04 LTS)
- Focal (20.04 LTS)
- Jammy (22.04 LTS)
- Noble (24.04 LTS)
- Questing (25.10)
